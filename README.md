#  PUNE Dairy Management System

A full-stack Milk Delivery Management System built using Flask (Python) backend and AngularJS frontend.

---

##  Features

* Admin login with authentication
* Customer management (Add / Edit / Delete)
* Staff management
* Milk products inventory
* Subscription management
* Order management
* Dashboard with statistics

---

##  Tech Stack

Backend:

* Python Flask
* SQLAlchemy
* SQLite

Frontend:

* AngularJS
* Bootstrap 5

---

##  Run Backend

cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python seed.py
python run.py

Backend runs at:

http://localhost:5000

---

##  Run Frontend

cd frontend
python -m http.server 4200

Open:

http://localhost:4200
